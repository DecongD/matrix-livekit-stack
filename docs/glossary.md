# Glossary

- Matrix: Open communication protocol and ecosystem.
- Synapse: Matrix homeserver implementation.
- Element: Web client for Matrix.
- LiveKit: WebRTC SFU for audio/video rooms.
- TURN: Relay protocol for NAT traversal.
- ICE: NAT traversal negotiation for WebRTC.
- Proxmox: Virtualization platform.
- Nginx: Reverse proxy and TLS terminator.
